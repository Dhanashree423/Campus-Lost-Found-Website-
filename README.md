# Lost & Found Portal


Campus Lost & Found Portal is a web application designed to help users report lost items and post found items.  

---

## Features
- Post a Lost Item
- Post a Found Item
- View all listings
- Clean and responsive user interface
- Simple form-based submission

---

## Technologies Used
- Html
- React
- TypeScript
- Vite
- Tailwind CSS

---

## How to Access the Website
Clone Repository (Recommended)

Requirements:
- Node.js (version 18 or above recommended)
- npm installed

Steps:

1. Clone the repository:
  Copy url by clicking on "CODE" Button above

2. Navigate into the project folder:
   cd repository-name

3. Install dependencies:
   npm install

4. Start development server:
   npm run dev

5. Open the link shown in terminal  
   (usually http://localhost:5173)
